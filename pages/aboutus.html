About the Site

Skills, knowledge and techniques used to make this website:
Title page animation:A combination of loops and arrays along with a series of images was used to create the animated effect of bullet holes appearing in the logo.
Sound effects:The background music was inserted into the website using the 'embed' tag, while a combination of arrays and javascript were used to achieve the sound effect on mouseover of the buttons.
Background:Once we selected the desired image to be used for the background, we separated the image into two parts according the top and bottom frames. The images were then aligned to top and bottom center and fixed in place so that they would stay in place when the page was scrolled.
Marquee:The 'marquee' tag along with javascript was used to create a scrolling poem.
Videos:Videos were embedded via YouTube to show gameplay and trailers of the games.
Weapons page:A page of weapons found in different games from the series was compiled using images found in the Call of Duty wiki.
Quiz:A quiz was created using a combination of loops, arrays, functions, if/else statements and javascript. The quiz is called 'What kind of soldier are you?' and uses information from the game series.
